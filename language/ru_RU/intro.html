Для кого этот плагин?

Profile Liveness Guard предназначен для администраторов галереи, которым нужно регулярно подтверждать, что владелец профиля по-прежнему контролирует подтверждённый номер телефона, связанный с корневым альбомом.

Что контролируется

  • Периодические проверки активности для владельцев альбомов без прав администратора
  • Ручная или плановая отправка SMS-подтверждений
  • Отслеживание льготного периода для неподтверждённых запросов
  • Автоматическое закрытие дерева альбомов после истечения срока
  • Восстановление видимости администратором и просмотр аудита

Рабочее замечание

LanguageSwitch может переключать PLG на любой установленный язык Piwigo. Теперь плагин поставляется с отдельными переводами PLG для английского (Великобритания), испанского, венгерского, словацкого, русского, украинского и упрощённого китайского.
