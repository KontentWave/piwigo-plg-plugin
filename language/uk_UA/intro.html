Для кого цей плагін?

Profile Liveness Guard призначений для адміністраторів галереї, яким потрібно регулярно підтверджувати, що власник профілю й надалі контролює підтверджений номер телефону, пов'язаний із кореневим альбомом.

Що контролюється

  • Періодичні перевірки активності для власників альбомів без прав адміністратора
  • Ручне або планове надсилання SMS-викликів
  • Відстеження пільгового періоду для непідтверджених викликів
  • Автоматична приватизація дерева альбомів після завершення строку
  • Відновлення видимості адміністратором і перегляд аудиту

Робоча примітка

LanguageSwitch може перемикати PLG на будь-яку встановлену мову Piwigo. Тепер плагін містить окремі переклади PLG для англійської (Велика Британія), іспанської, угорської, словацької, російської, української та спрощеної китайської.
